# Manual Data

This directory is for small curated context files that are useful but should not be scraped as a project dependency.

## Current Files

- `player_position_profiles.csv`: Transfermarkt-style position frequency context for player roles

## Rules

- Keep rows small and source-linked.
- Add `source_url` for every row.
- Treat this as context enrichment, not tactical evidence.
- Do not commit large exports or scraped website dumps.
